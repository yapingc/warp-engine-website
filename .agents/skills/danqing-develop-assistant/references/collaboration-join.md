# 受邀加入项目（协作开发）

当用户提供了邀请码和 appId，表示要加入他人的丹青约项目协作开发时，按以下流程处理：

## 1. 检查丹青约开发服务是否已安装

检查当前环境中是否已配置 `danqing-develop-mcp` 服务（检查对应 Agent 类型的配置文件中是否存在该服务配置）。

**如果已安装**：直接跳到第 3 步。

**如果未安装**：

## 2. 安装丹青约开发服务

按 Phase 1 的流程安装，但有以下简化：

- **安装范围**：默认全局安装，不需要询问用户
- **Chrome 调试服务**：一并默认安装
- 其余步骤不变（获取密钥、获取邮箱、写入配置）

安装完成后，引导用户重启：

> 开发服务已配置好。请先退出当前会话，重新启动后再把邀请码和应用名称发给我，我来帮您获取项目代码。

**重要**：此时必须让用户退出并重新启动，因为新安装的服务需要重启才能生效。用户重启后需要重新提供邀请码和 appId。

## 3. 使用邀请码加入项目

服务已就绪后，依次执行：

1. 调用 `join_app(invite_code="xxx")` 使用邀请码加入项目
2. 调用 `clone_project(app_id="xxx")` 克隆项目代码到本地
3. 执行项目依赖安装（`pnpm install`）
4. 配置域名（同 Phase 2 的域名配置流程）
5. 启动开发服务器并在 chrome-devtools 受控浏览器中打开页面

完成后告诉用户：

> 项目代码已获取并启动成功！您现在可以开始开发了。
